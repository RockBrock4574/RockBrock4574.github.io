Moveset Details

TLDR: no special moves or throws now, but there will be custom special moves. Everthing edited has fire effects

Currently, there is no change to the base special moves of Roy. This will be changed in the future, but, at this time, not only is the software I need to edit special moves not working, I still need to learn how to edit special move. If you know anything about making moveset mods for Smash Ultimate, special moves have way more code because of the status scripts tied to them, which require a different software to translate that is not working for me at the moment. The current moveset is moves that were ported and edited from other fighters, but the specials will be custom made. Any moves listed as ported from Roy/Chrom and Marth/Lucina have been edited so that the hitboxes are all the sweet spot of Roy or Marth. Also, there is no change to Roy's throws, and I don't plan on changing them, but that might change in the future. Everything that has been edited has fire effects, to match the nature of "flame breathing" in the manga and anime.

Smash Ultimate Rengoku Model and Voice

If you want to have a skin mod of Rengoku and a mod that gives you Rengoku's voice with my moveset mod, visit the Download section for a link to the mods.

To install either mod with my mod, download them and follow the steps and short video below on how to merge the folders.

  1. Download the mods and extract them.
  2. With the folders extracted, find the contents of the skin mod, select all of them, and copy them.
  3. Paste the contents into the proper folder of the moveset mod.
  4. Repeat for the voice mod.
  5. Drag the merged mod folder into your Smash Ultimate mods directory.
